## 权限认证模块

### 一、防止请求重复提交
1. 利用 AOP 切面在进入方法前拦截，通过 Redis 的 key-value 键值对存储，指定 key+url+消息头 来拼成字符串组成 key，使用 请求参数+时间 封装 map 对象赋值 value，当 key 不存在时，则为新的请求；若存在，则对请求参数以及请求的间隔时间进行判断是否重复提交。
